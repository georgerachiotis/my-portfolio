# Personal Portfolio

A modern developer portfolio built with React and Vite.

## Live Demo
https://georgerachiotis.github.io/my-portfolio/

## Features
- Responsive layout
- Project showcase
- Contact section
- Smooth scroll animations

## Tech Stack
- React
- Vite
- CSS

## Run Locally

Clone the project

git clone https://github.com/georgerachiotis/my-portfolio

Go to the project directory

cd my-portfolio

Install dependencies

npm install

Start the server

npm run dev
